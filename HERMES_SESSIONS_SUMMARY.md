## Рекомендации

### Для задачи 0: Умные заголовки
**Решение:** Использовать REST API

**Почему:**
- ✅ Безопаснее (валидация, транзакции)
- ✅ Автоматическое обновление FTS индексов
- ✅ Не нужен Python для создания сессий
- ✅ Проще тестировать

**Что делать:**
1. Удалить `scripts/create_session.py`
2. Добавить в `server.js`:
   - `POST /api/sessions` → прокси на Hermes REST API
   - `PATCH /api/sessions/:id/title` → прокси на Hermes REST API
3. Добавить в `src/lib/api.ts`:
   - `createSession(model, title)`
   - `updateSessionTitle(id, title)`
4. Реализовать LLM генерацию заголовка после первого ответа

### Для задачи 0.5: Управление сессиями (CRUD)
**Решение:** Использовать REST API

**Что делать:**
1. Добавить в `server.js`:
   - `DELETE /api/sessions/:id` → прокси на Hermes REST API
2. Добавить в `src/lib/api.ts`:
   - `deleteSession(id)`
3. Реализовать UI:
   - Кнопка удаления в SessionsPage
   - Кнопка удаления в ChatPage
   - Inline редактирование title

### Оставить как есть
- ✅ `scripts/save_messages.py` — нет REST API для сообщений
- ✅ Чтение сессий через SQLite — быстрее для больших объёмов
